# Neon Ninja

## Overview
Neon Ninja is a 2D swipe-slash runner game built in Unity. The player controls a ninja running through a neon-lit cyberpunk environment, using swipe gestures to dash, attack enemies, and avoid obstacles.

## Core Mechanics
- Continuous auto-runner movement.
- directional swipe controls (up, down, forward) to dash and attack.
- Neon aesthetic with fast-paced action.

## Setup Instructions
1. Open Unity Hub.
2. Select "Add project from disk" or "Open".
3. Navigate to and select the `NeonNinja` directory.
4. If this is a new setup, create a new 2D Core project here.
5. In Unity, go to **File -> Build Settings** and switch the platform to Android or iOS for mobile development. Configure the Game view resolution to a mobile portrait or landscape aspect ratio as desired for the runner.
